Introdução

ﾠﾠﾠﾠﾠﾠﾠﾠﾠ O que e folha de estilo?

ﾠﾠﾠﾠﾠﾠﾠﾠﾠ Uma folha de estilo é dita inline, quando as regras CSS são declaradas dentro da tag de abertura do elemento HTML com uso do atributo style.
ﾠﾠﾠﾠﾠﾠﾠﾠﾠ Estilo inline só se aplica a um elemento HTML.
ﾠﾠﾠﾠﾠﾠﾠﾠﾠ Ele perde muitas das vantagens das folhas de estilo pois mistura a apresentação com a marcação.
